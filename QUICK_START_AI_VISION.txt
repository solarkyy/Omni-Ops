## ğŸ› ï¸ Troubleshooting

### "Waiting for game frames..."
1. Ensure game is **fully loaded**
2. Click "**Start Game**" in menu
3. Wait 10 seconds for connection

### Commands not working?
1. Click on game window to focus
2. Try command again
3. Check terminal for errors

### AI analysis slow?
- Claude takes 2-5 seconds to analyze
- This is normal (API latency)
- Results cache while waiting

### "Port already in use"
1. Kill existing Python processes:
   ```bash
   taskkill /IM python.exe /F
   ```
2. Wait 5 seconds
3. Restart
